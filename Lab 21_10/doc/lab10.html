Getting Started 

You should have a Java project in Eclipse titled Lab 21_10. This html page (that you are reading) is included in the project in the doc directory, in the file lab10.html. The Java files you will use in this lab are in the src/pig and src/arraylist directories.

You will want to ensure that you have installed the EduRide tools; follow the directions in the link, or ask your TA.

If you have missing links then in the project properties (right click on "Lab 21_10" in the package explorer, select properties), edit the "Java Build Path" 

  • Add JUnit4 (click "add library") 
  • Add the eduride testing jar (click "add jar", and locate it in "lab21_10/lib/eduride-library.jar)
